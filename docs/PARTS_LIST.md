# Phase 1 部品リスト

**目的**：「15秒ごとに撮影 → BLE/WiFi で iPhone に転送 → Claude で説明文生成」が動く最小構成を組む。

価格は2026年5月時点・国内入手目安（円・税込）。為替や在庫で前後します。

---

## 1. メインボード

### Seeed XIAO ESP32-S3 Sense ★最重要
- **仕様**：ESP32-S3R8 + OV2640カメラ + PDMマイク + BLE/WiFi + LiPo充電IC内蔵、21×17.5mm
- **入手先**：
  - Seeed Studio 直販: https://www.seeedstudio.com/XIAO-ESP32S3-Sense-p-5639.html （約15USD）
  - Switch Science: https://www.switch-science.com/products/8253 （約2,200円）
  - 秋月電子（取り扱いあれば）
- **数量**：**2個推奨**（1個は破壊・実験用、1個は本番用）
- **小計**：約4,500円

OpenGlass と同じボード。カメラ・マイク・無線・充電が1枚に全部乗っていて、Phase 1〜2はこれ1枚で完結します。

## 2. 電池（LiPo）

### EEMB LP502030 3.7V 250mAh
- **OpenGlass採用品**。サイズ5×20×30mm、保護回路つき
- Amazon, AliExpress で入手可能（OpenGlassのREADMEにリンクあり）
- **代替**：同等サイズ・容量のLiPoならOK（保護回路ありを必ず選ぶ）
- **数量**：2個
- **小計**：約2,000円

> **警告**：保護回路なしのLiPoを顔横で使うのは絶対NG。過充電/短絡で発火します。

### JST PH 2.0mm コネクタ付きケーブル（電池側）
電池のコネクタが XIAO の BATT パッドに合わない場合、ケーブルで延長・変換するため。

## 5. 筐体（Phase 1ではクリップオン）

### 3Dプリント材料
- **PLA or PETG**：ザックリ試作用（クリップ形状の確認）
- **UV レジン（DLPプリンタ）**：精度・薄さが要る最終クリップ
- どちらも既に持っているなら追加購入不要

### 既存の眼鏡
- 普段使いの眼鏡に **クリップで取り外し可能** な形にすること。Phase 1で眼鏡本体に直接接着するのは避ける（破壊コストが高い）
